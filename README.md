## Tasks Overview

### Task 1 - Web Scraping

Scraped real estate property listings from MagicBricks.com for
4 Indian cities - Mumbai, Delhi, Bangalore and Ahmedabad.
Collected 80 properties with details like price, area and status.

### Task 2 - Exploratory Data Analysis

Performed in depth analysis on the scraped dataset to find insights
about property prices, areas and trends across cities. Found that
Mumbai is the most expensive city and size does not strongly
drive property prices in India.

### Task 3 - Data Visualization

Converted the EDA findings into 5 clear and meaningful charts
using matplotlib. Charts cover price comparison, area comparison,
status distribution and price per sqft across cities.
