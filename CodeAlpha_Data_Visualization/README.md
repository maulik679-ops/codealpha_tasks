# Task 3 - Data Visualization

## About

This project visualizes the insights found during the exploratory
data analysis of the real estate dataset. All charts are saved
as PNG images.

## Charts Created

- Average property price by city (bar chart)
- Average property area by city (bar chart)
- Property status distribution (pie chart)
- Average price per sqft by city (bar chart)
- Price vs area relationship (scatter plot)

## Key Insights from Charts

- Mumbai is the most expensive city by average price
- Bangalore has the largest properties on average
- Most properties in our dataset are ready to move in
- There is no strong relationship between price and area

## Libraries Used

- pandas - for loading and preparing data
- matplotlib - for creating all charts

- chart2_avg_area_by_city.png

1. Open Task3_Data_Visualization.ipynb in Google Colab
