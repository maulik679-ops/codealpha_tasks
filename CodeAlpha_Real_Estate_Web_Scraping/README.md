# Task 1 - Web Scraping

## About

This project scrapes real estate property listings from MagicBricks.com
for 4 Indian cities - Mumbai, Delhi, Bangalore and Ahmedabad.

## Data Collected

- Property Title
- Location
- Price
- Area (sqft)
- Status (Ready to Move / Under Construction)

## Cities Covered

- Mumbai
- Delhi
- Bangalore
- Ahmedabad

## Libraries Used

- requests - to send requests to the website
- BeautifulSoup - to extract data from HTML
- pandas - to store and save the data

## Output

- real_estate_data.csv - raw scraped dataset with 80 properties

## How to Run

1. Open Task1_Real_Estate_Web_Scraping.ipynb in Google Colab
2. Run all cells one by one
3. CSV file will be downloaded automatically
