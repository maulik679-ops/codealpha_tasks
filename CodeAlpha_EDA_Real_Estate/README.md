# Task 2 - Exploratory Data Analysis

## About

This project performs exploratory data analysis on the real estate
dataset collected in Task 1. The goal is to find meaningful insights
about property prices, areas and trends across Indian cities.

## What We Analyzed

- Average property price by city
- Average property area by city
- Property status distribution
- Price per sqft by city
- Most expensive and cheapest properties
- Correlation between price and area

## Key Findings

- Mumbai has the highest average property prices
- Bangalore has the largest average property area
- Price and area have a weak correlation meaning size does
  not strongly drive price in Indian real estate

## Libraries Used

- pandas - for data manipulation
- numpy - for numerical calculations
- matplotlib - for basic visualizations

## Output

- real_estate_clean.csv - cleaned dataset ready for visualization
